# Changelog

## 0.1.0 -- 2025-03-xx

### Added

- Cross-platform PTY child-process execution (`portable-pty`)
- Debounced filesystem watcher with diff-friendly events (`notify`)
- Read-only git fingerprint capture (HEAD, branch, dirty paths)
- Privacy-safe environment variable snapshot (allow-listed, BLAKE3-hashed)
- Lightweight resource profiler (RSS, page-faults, process count)
- Test adapter parsing for JUnit XML, pytest, vitest
- In-place payload redaction pipeline
- Session orchestration: start/stop with concurrent collectors
- 12 unit tests including adapter parsing
